## Requirements
### Requirement: KaTeX Support
The system MUST support LaTeX-style math rendering using KaTeX.

#### Scenario: Formula Rendering
Given text containing math delimiters
Then it MUST include the KaTeX assets
And it MUST render the formulas on page load.
